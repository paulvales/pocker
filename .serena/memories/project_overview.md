Pocker is a real-time Scrum Poker / planning poker app for distributed teams. It uses a Node.js + Socket.IO backend with a lightweight browser UI and optional PostgreSQL persistence for estimation history. Main entrypoint is index.js, which wires HTTP routes, Socket.IO handlers, room registry, logging, and history store. Core code areas: index.js; room-registry.js for room state and domain rules; src/handlers/socket.js for realtime events; src/routes/http.js for pages/assets/API; public/js/app.js for client behavior; estimation-history-store.js for PostgreSQL-backed history.
